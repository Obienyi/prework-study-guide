Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The attribute contains additional information of an element.
  • The parent element contains the child element.
  • The semantic HTML element represents the visible content accesible to variety of users.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector represents an element or attribute to which the CSS rule will apply.
  • A declaration is the CSS property we want to apply and the value of the property.
  • A wild card or * applies a rule to all the elements visible on the page.
  • A comma (,) applies a rule to two or more elements.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: stage files after a change.
  • git commit -m "commit message": commits a change.
  • git pull origin main: ensures no conflict between github and local repository.
